      21             :  *  \ingroup CoordinateMapsGroup
      22             :  *
      23             :  *  \brief Three dimensional map from the cube to a bulged cube.
      24             :  *  The cube is shaped such that the surface is compatible
      25             :  *  with the inner surface of Wedge3D.
      26             :  *  The shape of the object can be chosen to be cubical,
      27             :  *  if the sphericity is set to 0, or to a sphere, if
      28             :  *  the sphericity is set to 1. The sphericity can
      29             :  *  be set to any number between 0 and 1 for a bulged cube.
      30             :  *
      31             :  *  \details The volume map from the cube to a bulged cube is obtained by
      32             :  *  interpolating between six surface maps, twelve bounding curves, and
      33             :  *  eight corners. The surface map for the upper +z axis is obtained by
      34             :  *  interpolating between a cubical surface and a spherical surface. The
      35             :  *  two surfaces are chosen such that the latter circumscribes the former.
      36             :  *
      37             :  *  We make a choice here as to whether we wish to use the logical coordinates
      38             :  *  parameterizing these surface as they are, in which case we have the
      39             :  *  equidistant choice of coordinates, or whether to apply a tangent map to them
      40             :  *  which leads us to the equiangular choice of coordinates. In terms of the
      41             :  *  logical coordinates, the equiangular coordinates are:
      42             :  *
      43             :  *  \f[\textrm{equiangular xi} : \Xi(\xi) = \textrm{tan}(\xi\pi/4)\f]
      44             :  *
      45             :  *  \f[\textrm{equiangular eta}  : \mathrm{H}(\eta) = \textrm{tan}(\eta\pi/4)\f]
      46             :  *
      47             :  *  With derivatives:
      48             :  *
      49             :  *  \f[\Xi'(\xi) = \frac{\pi}{4}(1+\Xi^2)\f]
      50             :  *
      51             :  *  \f[\mathrm{H}'(\eta) = \frac{\pi}{4}(1+\mathrm{H}^2)\f]
      52             :  *
      53             :  *  The equidistant coordinates are:
      54             :  *
      55             :  *  \f[ \textrm{equidistant xi}  : \Xi = \xi\f]
      56             :  *
      57             :  *  \f[ \textrm{equidistant eta}  : \mathrm{H} = \eta\f]
      58             :  *
      59             :  *  with derivatives:
      60             :  *
      61             :  *  <center>\f$\Xi'(\xi) = 1\f$, and \f$\mathrm{H}'(\eta) = 1\f$</center>
      62             :  *
      63             :  *  We also define the variable \f$\rho\f$, given by:
      64             :  *
      65             :  *  \f[\rho = \sqrt{1+\Xi^2+\mathrm{H}^2}\f]
      66             :  *
      67             :  *  ### The Spherical Face Map
      68             :  *  The surface map for the spherical face of radius \f$R\f$ lying in the
      69             :  *  \f$+z\f$
      70             :  *  direction in either choice of coordinates is then given by:
      71             :  *
      72             :  *  \f[\vec{\sigma}_{spherical}: \vec{\xi} \rightarrow \vec{x}(\vec{\xi})\f]
      73             :  *  Where
      74             :  *  \f[
      75             :  *  \vec{x}(\xi,\eta) =
      76             :  *  \begin{bmatrix}
      77             :  *  x(\xi,\eta)\\
      78             :  *  y(\xi,\eta)\\
      79             :  *  z(\xi,\eta)\\
      80             :  *  \end{bmatrix}  = \frac{R}{\rho}
      81             :  *  \begin{bmatrix}
      82             :  *  \Xi\\
      83             :  *  \mathrm{H}\\
      84             :  *  1\\
      85             :  *  \end{bmatrix}
      86             :  *  \f]
      87             :  *
      88             :  *  ### The Cubical Face Map
      89             :  *  The surface map for the cubical face of side length \f$2L\f$ lying in the
      90             :  *  \f$+z\f$ direction is given by:
      91             :  *
      92             :  *  \f[\vec{\sigma}_{cubical}: \vec{\xi} \rightarrow \vec{x}(\vec{\xi})\f]
      93             :  *  Where
      94             :  *  \f[
      95             :  *  \vec{x}(\xi,\eta) =
      96             :  *  \begin{bmatrix}
      97             :  *  x(\xi,\eta)\\
      98             :  *  y(\xi,\eta)\\
      99             :  *  L\\
     100             :  *  \end{bmatrix}  = L
     101             :  *  \begin{bmatrix}
     102             :  *  \Xi\\
     103             :  *  \mathrm{H}\\
     104             :  *  1\\
     105             :  *  \end{bmatrix}
     106             :  *  \f]
     107             :  *
     108             :  *  ### The Bulged Face Map
     109             :  *  To construct the bulged map we interpolate between this cubical face map
     110             :  *  and a spherical face map of radius `radius_of_other_surface`, with the
     111             :  *  interpolation parameter being \f$s\f$, the `sphericity`.
     112             :  *  The surface map for the bulged face lying in the \f$+z\f$ direction is then
     113             :  *  given by:
     114             :  *
     115             :  *  \f[
     116             :  *  \vec{\sigma}_{+z}(\xi,\eta) = \left\{(1-s)L + \frac{sR}{\rho}\right\}
     117             :  *  \begin{bmatrix}
     118             :  *  \Xi\\
     119             :  *  \mathrm{H}\\
     120             :  *  1\\
     121             :  *  \end{bmatrix}
     122             :  *  \f]
     123             :  *
     124             :  *  We constrain L by demanding that the spherical face circumscribe the cube.
     125             :  *  With this condition, we have \f$L = R/\sqrt3\f$.
     126             :  *
     127             :  *  ### The General Formula for 3D Isoparametric Maps
     128             :  *  The general formula is given by Eq. 1 in section 2.1 of Hesthaven's paper
     129             :  *  "A Stable Penalty Method For The Compressible Navier-Stokes Equations III.
     130             :  *  Multidimensional Domain Decomposition Schemes" available
     131             :  *  <a href="
     132             :  *  http://citeseerx.ist.psu.edu/viewdoc/download?doi=10.1.1.699.1161&rep=rep1&type=pdf
     133             :  *  "> here </a>.
     134             :  *
     135             :  *  Hesthaven's formula is general in the degree of the shape functions used,
     136             :  *  so for our purposes we take the special case where the shape functions are
     137             :  *  linear, and define new variables accordingly.
     138             :  *
     139             :  *  We define the following variables for
     140             :  *  \f$\alpha, \beta, \gamma \in\{\xi,\eta,\zeta\}\f$:
     141             :  *
     142             :  *  \f[
     143             :  *  f^{\pm}_{\alpha} = \frac{1}{2}(1\pm\alpha)\\
     144             :  *  f^{\pm\pm}_{\alpha \ \beta} = \frac{1}{4}(1\pm\alpha)(1\pm\beta)\\
     145             :  *  f^{\pm\pm\pm}_{\alpha \ \beta \ \gamma} =
     146             :  *  \frac{1}{8}(1\pm\alpha)(1\pm\beta)(1\pm\gamma)
     147             :  *  \f]
     148             :  *
     149             :  *  The formula involves six surfaces, which we will denote by
     150             :  *  \f$\vec{\sigma}\f$, twelve curves, denoted by \f$\vec{\Gamma}\f$, and eight
     151             :  *  vertices, denoted by \f$\vec{\pi}\f$, with the subscripts denoting which
     152             :  *  face(s) these objects belong to. The full volume map is given by:
     153             :  *
     154             :  *  \f{align*}
     155             :  *  \phi(\xi,\eta,\zeta) = &
     156             :  *  f^{+}_{\zeta}\vec{\sigma}_{+\zeta}(\xi, \eta)+
     157             :  *  f^{-}_{\zeta}\vec{\sigma}_{-\zeta}(\xi, \eta)\\
     158             :  *  &+ f^{+}_{\eta}\vec{\sigma}_{+\eta}(\xi, \zeta)+
     159             :  *  f^{-}_{\eta}\vec{\sigma}_{-\eta}(\xi, \zeta)+
     160             :  *  f^{+}_{\xi}\vec{\sigma}_{+\xi}(\eta, \zeta)+
     161             :  *  f^{-}_{\xi}\vec{\sigma}_{-\xi}(\eta, \zeta)\\
     162             :  *  &- f^{++}_{\xi \ \eta}\vec{\Gamma}_{+\xi +\eta}(\zeta)-
     163             :  *  f^{-+}_{\xi \ \eta}\vec{\Gamma}_{-\xi +\eta}(\zeta)-
     164             :  *  f^{+-}_{\xi \ \eta}\vec{\Gamma}_{+\xi -\eta}(\zeta)-
     165             :  *  f^{--}_{\xi \ \eta}\vec{\Gamma}_{-\xi -\eta}(\zeta)\\
     166             :  *  &- f^{++}_{\xi \ \zeta}\vec{\Gamma}_{+\xi +\zeta}(\eta)-
     167             :  *  f^{-+}_{\xi \ \zeta}\vec{\Gamma}_{-\xi +\zeta}(\eta)-
     168             :  *  f^{+-}_{\xi \ \zeta}\vec{\Gamma}_{+\xi -\zeta}(\eta)-
     169             :  *  f^{--}_{\xi \ \zeta}\vec{\Gamma}_{-\xi -\zeta}(\eta)\\
     170             :  *  &- f^{++}_{\eta \ \zeta}\vec{\Gamma}_{+\eta +\zeta}(\xi)-
     171             :  *  f^{-+}_{\eta \ \zeta}\vec{\Gamma}_{-\eta +\zeta}(\xi)-
     172             :  *  f^{+-}_{\eta \ \zeta}\vec{\Gamma}_{+\eta -\zeta}(\xi)-
     173             :  *  f^{--}_{\eta \zeta}\vec{\Gamma}_{-\eta -\zeta}(\xi)\\
     174             :  *  &+ f^{+++}_{\xi \ \eta \ \zeta}\vec{\pi}_{+\xi +\eta +\zeta}+
     175             :  *  f^{-++}_{\xi \ \eta \ \zeta}\vec{\pi}_{-\xi +\eta +\zeta}+
     176             :  *  f^{+-+}_{\xi \ \eta \ \zeta}\vec{\pi}_{+\xi -\eta +\zeta}+
     177             :  *  f^{--+}_{\xi \ \eta \ \zeta}\vec{\pi}_{-\xi -\eta +\zeta}\\
     178             :  *  &+ f^{++-}_{\xi \ \eta \ \zeta}\vec{\pi}_{+\xi +\eta -\zeta}+
     179             :  *  f^{-+-}_{\xi \ \eta \ \zeta}\vec{\pi}_{-\xi +\eta -\zeta}+
     180             :  *  f^{+--}_{\xi \ \eta \ \zeta}\vec{\pi}_{+\xi -\eta -\zeta}+
     181             :  *  f^{---}_{\xi \ \eta \ \zeta}\vec{\pi}_{-\xi -\eta -\zeta}
     182             :  *  \f}
     183             :  *
     184             :  *
     185             :  *  ### The Special Case for Octahedral Symmetry
     186             :  *  The general formula is for the case in which there are six independently
     187             :  *  specified bounding surfaces. In our case, the surfaces are obtained by
     188             :  *  rotations and reflections of the upper-\f$\zeta\f$ face.
     189             :  *
     190             :  * We define the matrices corresponding to these transformations to be:
     191             :  *
     192             :  * \f[
     193             :  * S_{xy} =
     194             :  *  \begin{bmatrix}
     195             :  *  0 & 1 & 0\\
     196             :  *  1 & 0 & 0\\
     197             :  *  0 & 0 & 1\\
     198             :  *  \end{bmatrix},\
     199             :  *
     200             :  * S_{xz} =
     201             :  *  \begin{bmatrix}
     202             :  *  0 & 0 & 1\\
     203             :  *  0 & 1 & 0\\
     204             :  *  1 & 0 & 0\\
     205             :  *  \end{bmatrix},\
     206             :  *
     207             :  * S_{yz} =
     208             :  *  \begin{bmatrix}
     209             :  *  1 & 0 & 0\\
     210             :  *  0 & 0 & 1\\
     211             :  *  0 & 1 & 0\\
     212             :  *  \end{bmatrix}\f]
     213             :  *
     214             :  * \f[C_{zxy} =
     215             :  *  \begin{bmatrix}
     216             :  *  0 & 0 & 1\\
     217             :  *  1 & 0 & 0\\
     218             :  *  0 & 1 & 0\\
     219             :  *  \end{bmatrix},\
     220             :  *
     221             :  * C_{yzx} =
     222             :  *  \begin{bmatrix}
     223             :  *  0 & 1 & 0\\
     224             :  *  0 & 0 & 1\\
     225             :  *  1 & 0 & 0\\
     226             :  *  \end{bmatrix}\f]
     227             :  *
     228             :  * \f[N_{x} =
     229             :  *  \begin{bmatrix}
     230             :  *  -1 & 0 & 0\\
     231             :  *  0 & 1 & 0\\
     232             :  *  0 & 0 & 1\\
     233             :  *  \end{bmatrix},\
     234             :  *
     235             :  * N_{y} =
     236             :  *  \begin{bmatrix}
     237             :  *  1 & 0 & 0\\
     238             :  *  0 & -1 & 0\\
     239             :  *  0 & 0 & 1\\
     240             :  *  \end{bmatrix},\
     241             :  *
     242             :  * N_{z} =
     243             :  *  \begin{bmatrix}
     244             :  *  1 & 0 & 0\\
     245             :  *  0 & 1 & 0\\
     246             :  *  0 & 0 & -1\\
     247             :  *  \end{bmatrix}
     248             :  *  \f]
     249             :  *
     250             :  * The surface maps can now all be written in terms of
     251             :  * \f$\vec{\sigma}_{+\zeta}\f$ and these matrices:
     252             :  * <center>
     253             :  * \f$\vec{\sigma}_{-\zeta}(\xi, \eta) = N_z\vec{\sigma}_{+\zeta}(\xi, \eta)\\
     254             :  * \vec{\sigma}_{+\eta}(\xi, \zeta) = S_{yz}\vec{\sigma}_{+\zeta}(\xi, \zeta)\\
     255             :  * \vec{\sigma}_{-\eta}(\xi, \zeta) = N_yS_{yz}\vec{\sigma}_{+\zeta}(\xi,
     256             :  * \zeta)\\
     257             :  * \vec{\sigma}_{+\xi}(\eta, \zeta) = C_{zxy}\vec{\sigma}_{+\zeta}(\eta,
     258             :  * \zeta)\\
     259             :  * \vec{\sigma}_{-\xi}(\eta, \zeta) = N_xC_{zyx}\vec{\sigma}_{+\zeta}(\eta,
     260             :  * \zeta)\f$
     261             :  * </center>
     262             :  *
     263             :  * The four bounding curves \f$\vec{\Gamma}\f$ on the \f$+\zeta\f$ face are
     264             :  * given by:
     265             :  *
     266             :  * <center>
     267             :  * \f$\vec{\Gamma}_{+\xi,+\zeta}(\eta) = \vec{\sigma}_{+\zeta}(+1,\eta)\\
     268             :  * \vec{\Gamma}_{-\xi,+\zeta}(\eta) = \vec{\sigma}_{+\zeta}(-1,\eta)
     269             :  * = N_x\vec{\sigma}_{+\zeta}(+1, \eta)\\
     270             :  * \vec{\Gamma}_{+\eta,+\zeta}(\xi) = \vec{\sigma}_{+\zeta}(\xi,+1)
     271             :  * = S_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\\
     272             :  * \vec{\Gamma}_{-\eta,+\zeta}(\xi) = \vec{\sigma}_{+\zeta}(\xi,-1)
     273             :  * = N_yS_{xy}\vec{\sigma}_{+\zeta}(+1,\xi)\f$
     274             :  * </center>
     275             :  *
     276             :  * The bounding curves on the other surfaces can be obtained by transformations
     277             :  * on the \f$+\zeta\f$ face:
     278             :  *
     279             :  * <center>
     280             :  * \f$\vec{\Gamma}_{+\xi,-\zeta}(\eta) = N_z\vec{\sigma}_{+\zeta}(+1,\eta)\\
     281             :  * \vec{\Gamma}_{-\xi,-\zeta}(\eta) = N_z\vec{\sigma}_{+\zeta}(-1,\eta)
     282             :  * = N_zN_x\vec{\sigma}_{+\zeta}(+1,\eta)\\
     283             :  * \vec{\Gamma}_{+\eta,-\zeta}(\xi) = N_z\vec{\sigma}_{+\zeta}(\xi,+1)
     284             :  * = N_zS_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\\
     285             :  * \vec{\Gamma}_{-\eta,-\zeta}(\xi) = N_z\vec{\sigma}_{+\zeta}(\xi,-1)
     286             :  * = N_zN_yS_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\\
     287             :  * \vec{\Gamma}_{+\xi,+\eta}(\zeta) =
     288             :  * C_{zxy}\vec{\sigma}_{+\zeta}(+1,\zeta)\\
     289             :  * \vec{\Gamma}_{-\xi,+\eta}(\zeta) =
     290             :  * N_xC_{zxy}\vec{\sigma}_{+\zeta}(+1,\zeta)\\
     291             :  * \vec{\Gamma}_{+\xi,-\eta}(\zeta) = C_{zxy}\vec{\sigma}_{+\zeta}(-1,\zeta)
     292             :  * = C_{zxy}N_x\vec{\sigma}_{+\zeta}(+1,\zeta)\\
     293             :  * \vec{\Gamma}_{-\xi,-\eta}(\zeta) = N_xC_{zxy}\vec{\sigma}_{+\zeta}(-1,\zeta)
     294             :  * = N_xC_{zxy}N_x\vec{\sigma}_{+\zeta}(+1,\zeta)\f$
     295             :  * </center>
     296             :  *
     297             :  * Now we can write the volume map \f$\phi\f$ in terms of
     298             :  * \f$\vec{\sigma}_{+\zeta}\f$ only:
     299             :  * \f{align*}\phi(\xi,\eta,\zeta) = &
     300             :  * (f^{+}_{\zeta} + f^{-}_{\zeta}N_z)
     301             :  * \vec{\sigma}_{+\zeta}(\xi, \eta)\\
     302             :  * &+ (f^{+}_{\eta} + f^{-}_{\eta}N_y)
     303             :  * S_{yz}\vec{\sigma}_{+\zeta}(\xi, \zeta)\\
     304             :  * &+ (f^{+}_{\xi} + f^{-}_{\xi}N_x)
     305             :  * C_{zxy}\vec{\sigma}_{+\zeta}(\eta, \zeta)\\
     306             :  * &- (f^{+}_{\xi}+f^{-}_{\xi}N_x)
     307             :  * (f^{+}_{\eta}+f^{-}_{\eta}N_y)
     308             :  * C_{zxy}\vec{\sigma}_{+\zeta}(+1, \zeta)\\
     309             :  * &- (f^{+}_{\zeta}+f^{-}_{\zeta}N_z)\left\{
     310             :  * (f^{+}_{\xi}+f^{-}_{\xi}N_x)\vec{\sigma}_{+\zeta}(+1, \eta)+
     311             :  * (f^{+}_{\eta}+f^{-}_{\eta}N_y)S_{xy}\vec{\sigma}_{+\zeta}(+1, \xi)\right\}\\
     312             :  * &+ \frac{r}{\sqrt{3}}\vec{\xi}
     313             :  *  \f}
     314             :  *
     315             :  * Note that we can now absorb all of the \f$f\f$s into the matrix prefactors
     316             :  * in the above equation and obtain a final set of matrices. We define the
     317             :  * following *blending matrices*:
     318             :  *
     319             :  *  \f[
     320             :  *  B_{\xi} =
     321             :  *  \begin{bmatrix}
     322             :  *  0 & 0 & \xi\\
     323             :  *  1 & 0 & 0\\
     324             :  *  0 & 1 & 0\\
     325             :  *  \end{bmatrix},\
     326             :  *
     327             :  *  B_{\eta} =
     328             :  *  \begin{bmatrix}
     329             :  *  1 & 0 & 0\\
     330             :  *  0 & 0 & \eta\\
     331             :  *  0 & 1 & 0\\
     332             :  *  \end{bmatrix},\
     333             :  *
     334             :  *  B_{\zeta} =
     335             :  *  \begin{bmatrix}
     336             :  *  1 & 0 & 0\\
     337             :  *  0 & 1 & 0\\
     338             :  *  0 & 0 & \zeta\\
     339             :  *  \end{bmatrix}\\
     340             :  *
     341             :  *  B_{\xi\eta} =
     342             :  *  \begin{bmatrix}
     343             :  *  0 & 0 & \xi\\
     344             :  *  \eta & 0 & 0\\
     345             :  *  0 & 1 & 0\\
     346             :  *  \end{bmatrix},\
     347             :  *
     348             :  *  B_{\xi\zeta} =
     349             :  *  \begin{bmatrix}
     350             :  *  \xi & 0 & 0\\
     351             :  *  0 & 1 & 0\\
     352             :  *  0 & 0 & \zeta\\
     353             :  *  \end{bmatrix},\
     354             :  *
     355             :  *  B_{\eta\zeta} =
     356             :  *  \begin{bmatrix}
     357             :  *  0 & 1 & 0\\
     358             :  *  \eta & 0 & 0\\
     359             :  *  0 & 0 & \zeta\\
     360             :  *  \end{bmatrix}\\
     361             :  *
     362             :  *  B_{\xi\eta\zeta} =
     363             :  *  \begin{bmatrix}
     364             :  *  \xi & 0 & 0\\
     365             :  *  0 & \eta & 0\\
     366             :  *  0 & 0 & \zeta\\
     367             :  *  \end{bmatrix}
     368             :  *  \f]
     369             :  *
     370             :  *  Now we can write the volume map \f$\phi\f$ in these terms:
     371             :  *
     372             :  * \f{align*}
     373             :  * \phi(\xi,\eta,\zeta) = &
     374             :  * B_{\zeta}
     375             :  * \vec{\sigma}_{+\zeta}(\xi, \eta)\\& +
     376             :  * B_{\eta}
     377             :  * \vec{\sigma}_{+\zeta}(\xi, \zeta)+
     378             :  * B_{\xi}
     379             :  * \vec{\sigma}_{+\zeta}(\eta, \zeta)\\& -
     380             :  * B_{\xi \eta}
     381             :  * \vec{\sigma}_{+\zeta}(+1, \zeta)-
     382             :  * B_{\xi \zeta}
     383             :  * \vec{\sigma}_{+\zeta}(+1, \eta)+
     384             :  * B_{\eta \zeta}
     385             :  * \vec{\sigma}_{+\zeta}(+1, \xi)\\& +
     386             :  * B_{\xi \eta \zeta}
     387             :  * \vec{\sigma}_{+\zeta}(+1, +1)
     388             :  * \f}
     389             :  *
     390             :  * ### The Bulged Cube Map
     391             :  * We now use the result above to provide the mapping for the bulged cube.
     392             :  * First we will define the variables \f$\rho_A\f$ and \f$\rho_{AB}\f$, for
     393             :  * \f$A, B \in \{\Xi,\mathrm{H}, \mathrm{Z}\} \f$:
     394             :  * \f[
     395             :  * \rho_A = \sqrt{2 + A^2}\\
     396             :  * \rho_{AB} = \sqrt{1 + A^2 + B^2}
     397             :  * \f]
     398             :  * The final mapping is then:
     399             :  * \f[
     400             :  * \vec{x}(\xi,\eta,\zeta) = \frac{(1-s)r}{\sqrt{3}}
     401             :  * \begin{bmatrix}
     402             :  * \Xi\\
     403             :  * \mathrm{H}\\
     404             :  * \mathrm{Z}\\
     405             :  * \end{bmatrix} +
     406             :  * \frac{sr}{\sqrt{3}}
     407             :  * \begin{bmatrix}
     408             :  * \xi\\
     409             :  * \eta\\
     410             :  * \zeta\\
     411             :  * \end{bmatrix} + sr
     412             :  * \begin{bmatrix}
     413             :  * \xi & \Xi & \Xi\\
     414             :  * \mathrm{H} & \eta &\mathrm{H}\\
     415             :  * \mathrm{Z} & \mathrm{Z} & \zeta\\
     416             :  * \end{bmatrix}
     417             :  * \begin{bmatrix}
     418             :  * 1/\rho_{\mathrm{H}\mathrm{Z}}\\
     419             :  * 1/\rho_{\Xi\mathrm{Z}}\\
     420             :  * 1/\rho_{\Xi\mathrm{H}}\\
     421             :  * \end{bmatrix} - sr
     422             :  * \begin{bmatrix}
     423             :  * \Xi & \xi & \xi\\
     424             :  * \eta & \mathrm{H} &\eta\\
     425             :  * \zeta & \zeta & \mathrm{Z}\\
     426             :  * \end{bmatrix}
     427             :  * \begin{bmatrix}
     428             :  * 1/\rho_{\Xi}\\
     429             :  * 1/\rho_{\mathrm{H}}\\
     430             :  * 1/\rho_{\mathrm{Z}}\\
     431             :  * \end{bmatrix}
     432             :  * \f]
     433             :  *
     434             :  * In the case where the same coordinates are used for the cube and the sphere,
     435             :  * we have \f$\xi = \Xi\f$, etc. In this case, the formula reduces further.
     436             :  * It is given by:
     437             :  *
     438             :  * \f[
     439             :  * \vec{x}(\xi,\eta,\zeta) =
     440             :  * \left\{
     441             :  * \frac{r}{\sqrt{3}}
     442             :  * + sr
     443             :  * \left(
     444             :  * 1/\rho_{\mathrm{H}\mathrm{Z}}+
     445             :  * 1/\rho_{\Xi\mathrm{Z}}+
